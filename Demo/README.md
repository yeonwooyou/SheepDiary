# Demo — AI 일기 생성(목업) 실행

이 폴더는 **AI 일기 생성 흐름을 설명하기 위한 데모**입니다.  
실제 LLM/Vision API 호출은 포함하지 않고, **목업 데이터**로 프롬프트 구성 → 결과 출력까지 재현합니다.

## 실행 방법

```bash
cd "Demo"
python3 diary_generator.py
```

## 실행 결과
- 터미널에 아래가 순서대로 출력됩니다.
  - 생성된 프롬프트
  - (Mocked) API 호출 안내 문구
  - 최종 생성된 목업 일기

## 참고
- 이 데모는 외부 API 키가 필요 없습니다.
